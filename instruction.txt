Исходные данные:

* тесты написаны на языке Python (ver. 3.9.5)
* среда тестировани Pytest
* операционная система Windows



Инструкция по развертыванию тестов:

- файл с тестами 'test_todo.py' (далее 'файл')
- открываем файл:
	- в 39 строке вместо 'https://...' прописываем ссылку на ресурс/сайт (далее 'ресурс'), подлежащий тестированию
	- в 69 строке вместо 'SECRET_KEY' прописываем ключ, присвоенный ресурсом при первой аутентификации пользователя
	- в 70 строке вместо 'token' прописываем токен, выданный ресурсом при первой аутентификации пользователя
- открываем "командную строку"
- запускаем файл с помощью команды 'pytest test_todo.py'
